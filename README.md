## Development

### Install Dependencies

```shell
npm install
```

### Available Commands

Run the Dev Server

```shell
npm run dev
```

Check for, and optionally fix, code problems

```shell
npm run lint
```

```shell
npm run lint:fix
```

Build The Site

```shell
npm run build
```

Build The Site Without Linting

```shell
npm run build:ci
```

Start The Production Server

```shell
npm run start
```

Export Static HTML

```shell
npm run export
```
